=======================================
How to use the 5-text_indentation modul
=======================================

This modul defines the function ``text_indentation(text):``

Usage: text_indentation(text)
=============================

# function that prints a text with 2 new lines after each of these
characters: `.`, `?` and `:`
# text must be a string, otherwise raise a TypeError exception with the
message "text must be a string"

Example of successful tests:
============================

first import a modul
	>>> text_indentation = __import__('5-text_indentation').text_indentation

print text strings:
>>> text_indentation("""Lorem ipsum dolor sit amet, consectetur adipiscing elit. """)
Lorem ipsum dolor sit amet, consectetur adipiscing elit.
<BLANKLINE>

prints empty string
>>> text_indentation("")
